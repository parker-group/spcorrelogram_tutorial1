Spatial Correlogram Tutorial

This repo mirrors the minimal structure of variogram_tutorial1 but focuses on spatial correlograms (Moran’s I and Geary’s C across distance bins).

How to run

  1. Open correlogram_tutorial1.rmd in R (RStudio optional).
  2. Install required packages once: 
    install.packages(c("sf","sp","spdep","ggplot2"))
  3. Knit the Rmd to HTML. The knitted file (e.g., correlogram_tutorial1.html) can be viewed locally or published with GitHub Pages.

Need a README or extras? Keep this repo minimal like the original, or add files in a separate branch.
